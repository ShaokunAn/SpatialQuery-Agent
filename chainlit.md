# SpatialQuery Agent

Welcome! This agent helps you analyze spatial transcriptomics data using natural language.

## Getting Started

1. **Load your data** — Type: `/path/to/your/data.h5ad`
2. **Ask a question** — e.g., "Find enriched neighborhoods around Endothelium cells"
3. **Follow suggestions** — The agent will guide you through downstream analyses

## Supported Analyses

- **Motif Enrichment** — Discover enriched cell-type neighborhoods
- **Differential Expression** — Find DE genes between spatial niches
- **Gene Co-variation** — Analyze gene-gene correlations in spatial context
- **Parameter Sweep** — Explore sensitivity to analysis parameters
- **Visualization** — Spatial plots, heatmaps, and more

Type `agent:` before your message to use the analysis agent, or `chat:` for Q&A about the SpatialQuery API.
